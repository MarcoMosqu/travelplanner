# Investor Match Website

**Investor Match** is a simple website designed to connect startups and businesses with the right investors. It helps entrepreneurs showcase their ideas, find funding, and build meaningful partnerships.

## Features

- About section explaining the platform
- Services section with investor matchmaking and business support
- Join section encouraging entrepreneurs to sign up
- Navigation bar for easy access
- Footer with copyright

## How to Use

1. Open `index.html` in any web browser.
2. Explore the sections: About, Services, Join.
3. Modify content as needed or link additional pages.
4. Add your own styles in `styles.css`.
